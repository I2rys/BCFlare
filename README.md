## Installation
Github:

    git clone https://github.com/I2rys/BCFlare

NPM Packages:

    npm i puppeteer-extra-plugin-stealth && npm i puppeteer-extra
    
This NPM Package:

    npm i bcflare
    

## Usage
Get:

    BCFlare.get(url, headers, useragent)
    
Post:

    BCFlare.post(url, post_body, headers, useragent)
